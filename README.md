# Цифровая Библиотека (E-Library)

Каталог книг онлайн, который позволяет читать прямо в браузере. Сайт полностью автономен: работает как по ссылке онлайн, так и без доступа к интернету.

**[Перейти в онлайн версию](https://0rcb0n3-lab.github.io/e_library/)** — читайте прямо сейчас в один клик.

## Руководство для разработчиков

При создании сайта использовалась генерация статики с помощью ***Python*** и ***Jinja2***. Ниже инструкции по локальной настройке и управлению.

### Требования

* **Python** версии 3.14

### Быстрый старт

1. **Склонируйте репозиторий на свой компьютер:**

2. **Установите необходимые библиотеки:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Запустите локальный сервер разработки:**

   ```bash
   python render_website.py
   ```

   После запуска откройте в браузере адрес `http://localhost:5500` (или `http://127.0.0.1:5500` ).

   >Любые изменения в шаблоне или базе данных будут мгновенно обновлять страницу.

### Параметры командной строки

Скрипт поддерживает гибкую настройку генерации страниц с помощью аргументов:

>`-db`, `--data_base` — Путь к JSON-файлу с метаданными книг (по умолчанию: `meta_data.json`).

>`-bp`, `--books_on_pages` — Лимит количества книг, отображаемых на одной странице (по умолчанию: `10`).

**Примеры кастомизации сборки:**

```bash
# Сборка каталога с увеличенным количеством книг на страницу
python render_website.py -bp 20

# Подключение альтернативной базы данных
python render_website.py -db custom_books.json

# Комбинированный запуск с полной настройкой
python render_website.py --data_base test_library.json --books_on_pages 15
```

## Структура репозитория

```text
e-library/
├── render_website.py   # CLI-скрипт сборки сайта и запуска livereload
├── template.html       # Универсальный HTML-шаблон (Jinja2)
├── meta_data.json      # Структурированная база данных каталога
├── index.html          # Главная страница (точка входа)
├── pages/              # Автоматически генерируемые страницы пагинации
├── media/              # Хранилище обложек и файлов книг
└── static/             # Локальные стили (CSS) и скрипты (JS) Bootstrap
```

### Спецификация файла `meta_data.json`
База данных книг должна представлять собой массив объектов следующей структуры:

```json
[
  {
    "title": "Название произведения",
    "author": "Имя Автора",
    "genres": "Фантастика, Приключения",
    "img_src": "img/cover.jpg",
    "book_path": "media/books/book.txt"
  }
]
```

## Зависимости

Для работы скрипта используются:
* `jinja2==3.1.6` — Шаблонизатор для генерации HTML.
* `livereload==2.7.1` — Локальный сервер с автообновлением страниц в реальном времени.
* `more-itertools==11.1.0` — Разбиение списка книг на страницы и строки (chunking).
